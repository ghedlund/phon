Look Ahead/Behind

Look ahead/behind in phonex allows matching based on adjacent elements. The elements matched by the look ahead-beind groups will not be included in the returned value. Look ahead groups will start with the sequence (?> while look behind groups will start with the sequence (?<.

One case where this feature is useful is when searching for patterns medially within transcriptions. For example, to seach for all word-medial consonants:
(?<\w)\c(?>\w)
This example has main parts:
  1. (?<\w) look behind the current element and match if it is a consonant or vowel
  2. \c match if the current element is a consonant
  3. (?>\w) look ahead of the current element and match if it is a consonant or vowel
