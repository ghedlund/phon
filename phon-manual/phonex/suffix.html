suffix

Description of the suffix supplementary matcher.

The prefix supplementary matcher queries the prefix section of a phone. It accepts a single parameter which can be either:
  • A single character
  • A feature set matcher
  • A phone class matcher

The matcher will match if any of the diacritics of the phone match the provide argument.

Search for a phone with the suffix diacritic \u02b0
.:suffix("\u02b0\")
