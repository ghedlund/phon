PhonTalk

PhonTalk is a plug-in for the Phon application. As of Phon 3.1.4, PhonTalk is included with Phon and includes the following features:

  • Import Project wizard

  • Export Project wizard

  • Ability to open CHAT (.cha) and TalkBank (.xml) files as Phon sessions. CHAT and TalkBank files may be included in Phon corpus folders without need for conversion. When saving the session an option will be presented to the user to save in Phon format or in the original format.
  • Abililty to save Phon sessions as CHAT (.cha) or TalkBank (.xml) files using the File > Save as menu in the session editor.

Source code can be found at https://github.com/phon-ca/phontalk.

To report an issue, please go to https://github.com/phon-ca/phontalk/issues.
