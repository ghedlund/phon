PhonTalk

PhonTalk is an application for converting between Phon project and the Talkbank XML format. Files processed in the application must be generated using the chatter application available at http://www.talkbank.org/software. CHAT corpora can be downloaded from the CHILDES website at http://childes.psy.cmu.edu/data/.
