      • View: Media Player
        • About Media
        • Open Media

          Open a media file which has already been linked to a session.

        • Go to a Specific Playback Time

          Jump to a specific playback time (either user-specified, at the end of segmented media, or at the end of the last segment for a certain speaker.

        • Move Media Position with Record

          Choose whether the current media position should move to align with the start of each segment as you navigate between records.

        • Play Adjacency Sequence

          Play the current record and the following one, regardless of Speaker.

        • Play Current Segment

          Play the media segment specified in the currently visible record.

        • Play Current Speech Turn

          Play adjacent utterances produced by the current speaker, starting with the current record.

        • Play Custom Segment

          Play a user-defined segment of your media file.

        • Export Media from Media Player

          Export clips of your media file from the Media Player view.

        • Take Snapshot of Media

          Take a snapshot of linked media at the current viewpoint.
