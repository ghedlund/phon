Participant Filter

Limit the results based on participant information.


  • To filter by participant names, enter one or more names in the Participant names field, separated by comma if applicable.
  • To filter by age, use the context menu to select equal to, less than, or greater than and enter a name in the first Age field (using the format yy;mm.dd).
  • To filter using an age range, select an operator using the context menu, and specify the relevant options and age in the second Age field.
