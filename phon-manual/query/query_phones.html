Query: Phones

The Phones query is used to query data contained within IPA Target and IPA Actual tiers.



Parameters

  • Search Tier: IPA Target or IPA Actual.

  • Search by: Group, Word and optionally by syllable. This options defines the search domain for the query.

  • Expression type: Plain text, Regular Expression, Phonex, Stress Pattern, CGV Pattern

  • Expression: query expression

    • Case sensitive (not applicable for phonex expressions)

    • Exact match

    • Allow overlapping matches (phonex only)

Aligned Phones

The phones aligned with the queried phones can be added to query results. If 'include aligned phones' is selected the aligned results may be filtered using the IPA Target Matcher and IPA Actual Matcher expressions.
