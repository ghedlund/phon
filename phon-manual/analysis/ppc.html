Percent Phones Correct (PPC)

The PPC analysis will calculate the percent phones/consonants/vowels correct in the sampled data.

Data Preparation

Session data is queried by word for each IPA Target and IPA Actual tuple. Proper syllabification and phone alignment is required for accurate results. All participants should have unique names/identification numbers and a speaker should be assigned to all records.

Parameters

The PPC report can be restricted to different sets of phones. Several pre-defined sets have been defined and are available by using the radio buttons displayed in the PPC form (see below.) The following pre-defined report types are available:
  1. Percent phones correct - all phones.
  2. Percent consonants correct - all consonants (i.e., PCC)
  3. Percent vowels correct
  4. Percent singleton consonants correct - singleton consonants.
  5. Percent cluster consonants correct - cluster consonants. There are three report sub-types available when this option is selected:
    1. All clusters
    2. Tautosyllabic clusters
    3. Heterosyllabic clusters
  6. Percent onset consonants correct. There are three report sub-types available when this option is selected:
    1. All onsets
    2. Singleton onsets
    3. Cluster onsets
  7. Percent coda consonants correct. There are three report sub-types available when this option is selected:
    1. All codas
    2. Singleton codas
    3. Cluster codas
  8. Percent nucleus phones correct. There are three report sub-types available when this option is selected:
    1. All nuclei
    2. Monophthongs
    3. Complex vocoids
  9. Percent correct (custom)

When 'Percent correct (custom)' is selected a custom Phonex expression may be entered to create a custom phone set. See Example Phones Queries for some useful examples.

Other parameters:
  1. PPC-r - phones marked with the distorted diacritic will be considered correct (default: no)
  2. Ignore diacritics - ignore diacritics when comparing phones (default: no)
  3. Include PPC w/o epenthesis 'PPC (NoEpen)' (see below)
  4. Group Options
  5. Word Options
  6. Syllable Options
